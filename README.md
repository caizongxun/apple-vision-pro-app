# Apple Vision Pro App

A visionOS application built with SwiftUI and RealityKit.

## Requirements

- Xcode 15+
- visionOS SDK 1.0+
- Apple Vision Pro or visionOS Simulator

## Project Structure

```
apple-vision-pro-app/
├── App/
│   ├── AppEntry.swift          # App entry point
│   ├── ContentView.swift       # Main content view
│   ├── ImmersiveView.swift     # Immersive space view
│   └── Assets.xcassets/
├── README.md
└── .gitignore
```

## Getting Started

1. Clone the repository
2. Open `apple-vision-pro-app.xcodeproj` in Xcode
3. Select the visionOS simulator or device
4. Build and run (`Cmd + R`)

## Features

- [ ] Window scene
- [ ] Immersive space
- [ ] 3D content with RealityKit
- [ ] Hand gesture interaction
